# Run on Android

Flutter ilovasini ulangan Android qurilmada ishga tushirish.

## Qadamlar

// turbo
1. Ulangan qurilmalarni tekshirish:
```bash
flutter devices
```

// turbo
2. Android qurilmada ishga tushirish:
```bash
flutter run -d android
```

## Foydali buyruqlar

- `r` - Hot reload
- `R` - Hot restart
- `q` - Dasturni to'xtatish

## Muammo bo'lsa

```bash
flutter clean
flutter pub get
flutter run -d android
```
