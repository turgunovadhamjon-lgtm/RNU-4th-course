# 🎓 RNU 4th Course - Thesis Work Repository
**Черновики дипломных работ / Diploma Work Drafts / Bitiruv malakaviy ishi qoralamalari**

Welcome to the central repository for the 4th-year students. This workspace is designated for uploading drafts of diploma works and applications, allowing professors and tutors to review, guide, and leave professional feedback.

---

## 📁 Repository Structure

We have organized the repository into two main areas:
- **`application/`** — Please upload any application forms, proposal texts, or supplementary documents here.
- **`diploma_work/`** — Please upload the chapters, code components, or full drafts of your thesis here.

**Important:** Make sure to create a sub-folder with your Name and Surname inside these directories so things remain organized! (e.g., `diploma_work/John_Doe/`)

---

## 🚀 How to Upload Your Work (Git Commands)

### 🇬🇧 English: Step-by-Step Guide
1. **Clone the repository:**
   ```bash
   git clone https://github.com/turgunovadhamjon-lgtm/RNU-4th-course.git
   cd RNU-4th-course
   ```
2. **Pull the latest changes** (always do this before uploading!):
   ```bash
   git pull origin main
   ```
3. **Copy your files into your personal folder** within `application` or `diploma_work`.
4. **Stage and commit your files:**
   ```bash
   git add .
   git commit -m "Add John Doe's chapter 1 draft"
   ```
5. **Push to the repository:**
   ```bash
   git push origin main
   ```

### 🇷🇺 Русский: Пошаговая инструкция для студентов
1. **Клонируйте репозиторий себе на компьютер (один раз):**
   ```bash
   git clone https://github.com/turgunovadhamjon-lgtm/RNU-4th-course.git
   cd RNU-4th-course
   ```
2. **Скачайте последние обновления** (обязательно перед каждой новой загрузкой!):
   ```bash
   git pull origin main
   ```
3. **Скопируйте ваши файлы** в свою личную папку внутри `application` или `diploma_work`.
4. **Добавьте и сохраните (commit) ваши файлы в Git:**
   ```bash
   git add .
   git commit -m "Добавлен черновик главы 1 - Иван Иванов"
   ```
5. **Отправьте изменения на сервер (на проверку):**
   ```bash
   git push origin main
   ```

## 👨‍🏫 Tutors and Professors: Review Guidelines

Dear Professors, you can review the uploaded materials and leave feedback using the following methods:
1. **Pull Requests (Recommended):** If students use feature branches, you can review their Pull Requests, add inline comments, and request changes before approving.
2. **Github Issues:** You can open an **Issue** referencing a student's specific file or folder to discuss necessary corrections.
3. **Direct Feedback & Commits:** You can clone the repository, add a `.txt` or `.md` file with feedback inside the student's personal folder, and push it directly so they can see your notes.

*Ensure all feedback is constructive and helps students finalize their thesis effectively.*
